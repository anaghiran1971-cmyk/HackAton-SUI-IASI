# Glosar Sui

- Object: unitate de date fundamentală pe Sui.
- Ownership: cine “deține” un obiect.
- Shared object: obiect utilizat de mai multe tranzacții.
- Package: bundle de module Move.
- Module: unitate de cod Move.
- Entry function: funcție apelabilă din tranzacție.
- Gas: taxă de execuție.
- Digest: identificator tranzacție.
