# Securitate Wallet Copilot

Copilotul NU trebuie să:
- ceară seed phrase / private key
- semneze tranzacții în locul userului
- trimită fonduri fără confirmare în wallet

Copilotul trebuie să:
- genereze intenții (draft)
- evalueze risc
- explice tranzacții
- blocheze execuția la risc ridicat / lipsă confirmare
