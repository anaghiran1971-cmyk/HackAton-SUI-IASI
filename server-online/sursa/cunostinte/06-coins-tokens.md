# Coins / Tokens pe Sui

## Coin vs Token
În practică “coin” este un tip Move care reprezintă o cantitate fungibilă.
CoinType tipic: `0x2::sui::SUI`.

## Operațiuni comune
- split coins: separi o sumă din coin-ul de gas
- merge coins: unești coin-uri
- transfer: trimiți coin către destinatar

## Atenție la unități
- 1 SUI = 1_000_000_000 MIST
- Tranzacțiile folosesc de obicei MIST intern.
