# Move Patterns (utile pentru audit și înțelegere)

## Capability pattern
Unele acțiuni cer “capabilities” (permisiuni) reprezentate ca obiecte.

## Admin keys / privileged functions
Caută:
- funcții care pot mint/burn
- funcții care pot schimba parametri critici
- roluri/permisiuni centralizate

## Best practice
- verifică ce entry functions există
- verifică cine poate apela funcțiile sensibile
