# NFT pe Sui (conceptual)

## Ce este un NFT
Un NFT este de regulă un obiect (sau set de obiecte) care conține:
- metadate (nume, descriere, URI imagine)
- owner (adresă)

## Colecții
O colecție poate fi reprezentată diferit, dar util e să ai:
- packageId (unde e codul Move)
- creator
- marketplace / sursă
- semnal “verificat” (intern sau dintr-o listă)

## Riscuri comune
- colecții fake (nume similar)
- contracte noi fără istoric
- metadate lipsă sau dubioase
