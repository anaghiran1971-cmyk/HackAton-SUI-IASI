# Introducere în Sui

Sui este un blockchain orientat pe obiecte. Modelul de date principal este "object".

## Object ownership
- Owned objects: aparțin unui owner (adresă).
- Shared objects: pot fi accesate de mai multe tranzacții, cu reguli specifice.
- Immutable objects: nu se schimbă.

## Gas & execuție
Tranzacțiile consumă gas. Pe Sui, tranzacțiile pot executa apeluri Move și pot muta/crea/modifica obiecte.
