# Recomandări pentru demo (hackathon)

## Demo flow recomandat
1) Prompt: "Trimite lui marcel 0.01 SUI"
2) Parse intenție -> rezolvă destinatar -> risc automat -> confirmare wallet
3) Explică tranzacția cu txDigest
4) Arată "safe-by-default": încearcă 5 SUI când limita e 1 SUI
