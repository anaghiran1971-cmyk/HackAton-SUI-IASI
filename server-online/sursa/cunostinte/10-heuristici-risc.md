# Heuristici de risc (pragmatic)

## Transfer
Semnale:
- destinatar nou / necunoscut
- sumă mare (peste limită sau procent mare din balanță)
- multe transferuri repetate într-un timp scurt

## Token
Semnale:
- proiect/contract nou, fără istoric
- lipsă metadate
- sursă necunoscută

## NFT
Semnale:
- colecție necunoscută, neverificată
- nume similar cu o colecție celebră (spoof)
- metadate incomplete
